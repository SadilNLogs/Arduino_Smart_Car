# Arduino Bluetooth Controlled 4WD Smart Car
## Project Overview
### This project is a Bluetooth-controlled 4-wheel robotic car built using an Arduino Uno, HC-05 Bluetooth module, L298N Motor Driver, and four DC geared motors. The vehicle can move forward, backward, left, right, and stop through commands sent from an Android Bluetooth controller application.

The project demonstrates embedded systems programming, motor control, wireless communication, and robotics fundamentals.
